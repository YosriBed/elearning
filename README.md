# eLearning

WIP

A demo eLearning app.

* Client:  [React.js](https://reactjs.org/)
* Server: [Node.js](https://nodejs.org/en/)
* Database: [Mongodb](https://www.mongodb.com/)


## Installation

### client
```
cd client/
cp .env.example .env #don't forget to change the envrionment variables
npm install
npm start
```
### server
```
cd server/
cp .env.example .env #don't forget to change the envrionment variables
npm install
npm start
```
